## 📖 About the Game

**Final Destination** is a story-driven survival game where the player makes choices that affect the outcome of the story. Each decision can lead to survival, death, or different story paths.
 
The game features multiple scenes, and each scene presents the player with different choices. Depending on the selected option, the story progresses, branches into different scenarios, or ends in a game over.

---

## 🎮 Core Gameplay

```
1. 🎭 Scene-based gameplay system
2. 🔠 Multiple choices per scene (A, B, C, D)
3. 🎋 Branching story paths
   ❌ Game over system with custom death messages
4. 💀 Ending scene with multiple outcomes
5. ➡️ Return to main menu system
```

---
##  🏫Concepts Used
 
- Object-Oriented Programming (OOP)
  - Encapsulation (Scene and Choice classes)
  - Abstraction (Game logic separated in GameEngine)
  - No inheritance yet because I am still planning on changing the scenes
  - Same as inheritance, no poly yet
- Conditional statements
- Loops (specifically while loop)
- ArrayList for dynamic choices (I got stuck with two choices (A & B) and I wanted to have more choices)
